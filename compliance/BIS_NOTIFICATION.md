# BIS Open-Source Encryption Notification

Publicly available open-source encryption software qualifies for the
notification procedure under EAR §742.15(b) and §734.3(b)(3). Send the
following one-time email **before or at the time of publication**:

**To:** crypto@bis.doc.gov
**Cc:** enc@nsa.gov
**Subject:** Notification of publicly available open-source encryption software — Lockjaw

> In accordance with 15 CFR §742.15(b) of the Export Administration
> Regulations, this is a notification that publicly available open-source
> encryption source code is published at the following location:
>
> https://github.com/kevinbdill/lockjaw
>
> Product name: Lockjaw
> Description: Open-source file encryption utility using libsodium
> (XChaCha20-Poly1305 authenticated encryption, Argon2id key derivation).
>
> Contact: [your name and email]

That is the entire requirement. Keep a copy of the sent email. If the
repository URL changes, send an updated notification.

*This is a compliance convenience note, not legal advice.*
